# 🚀 Admin Dashboard (MERN Stack)

A modern full-stack admin dashboard built with the MERN stack, featuring user management, authentication, and real-time UI updates.

---

## ✨ Features

- 🔐 JWT Authentication (Admin only access)
- 👥 User Management (Create, Edit, Delete)
- 🛡 Role-based Access Control (Admin / User / Editor)
- 🚦 User Status Control (Active / Suspended)
- 🖼 Avatar Upload (Multer + Cloudinary)
- ⚡ Optimistic UI Updates (React Query)
- 📊 Dashboard Stats & UI Components
- 🎨 Responsive UI with Tailwind CSS

---

## 🧰 Tech Stack

### Frontend

- React.js
- Tailwind CSS
- React Query
- Axios

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer + Cloudinary

## ⚙️ Installation

### 1. Clone the repo

```
git clone https://github.com/your-username/admin-dashboard-mern.git
```

### 2. Install dependencies

#### Backend

```
cd admin-dashboard-backend
npm install
```

#### Frontend

```
cd premium-dashboard
npm install
```

---

### 3. Setup environment variables

Create a `.env` file in the server folder:

```
PORT=8000
MONGO_URI=your_mongo_uri
JWT_SECRET=your_secret
CLOUD_NAME=your_cloud_name
CLOUD_API_KEY=your_api_key
CLOUD_API_SECRET=your_api_secret
```

---

### 4. Run the app

#### Backend

```
npm run watch
```

#### Frontend

```
npm run dev
```
